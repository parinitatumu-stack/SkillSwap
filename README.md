# SkillSwap 🌐
### Campus Skill Exchange Network Hub

A web platform where university students can **offer skills they know** and **request skills they want to learn** — like a barter system for knowledge. Built as part of the FWD Project Expo at KL University Bachupally.

🔗 **Live Demo:** [skillswap-skill-exchange-network.netlify.app](https://skillswap-skill-exchange-network.netlify.app)

---

## 📌 Features

- 🔐 Google Sign-In — only campus students can access
- 🤝 Real-time skill matching between students
- 📹 Live video calls via Jitsi Meet API when matched
- 🔍 Search and filter skills by category
- 📱 Fully responsive — works on mobile, tablet, desktop
- 🌙 Dark / Light mode toggle

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Semantic structure of the app |
| CSS3 | Styling, Flexbox, Grid, animations, media queries |
| JavaScript (ES6+) | Logic, DOM manipulation, array methods |
| Firebase Auth | Google Sign-In authentication |
| Firebase Firestore | Real-time database |
| Jitsi Meet API | In-app live video calls |
| Netlify | Deployment and hosting |
